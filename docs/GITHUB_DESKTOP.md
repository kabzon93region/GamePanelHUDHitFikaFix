# GitHub Desktop — quick start / Быстрый старт

**EN** | [RU](#русский)

## English

1. **GitHub Desktop** → **File** → **Add local repository**
2. Choose folder: `...\github-repos\GamePanelHUDHitFikaFix`
3. If prompted “not a git repo” → **create a repository** in this folder
4. Initial commit message: `Initial release v1.0.0 — Fika hit marker fix for GamePanelHUD`
5. **Publish repository** (public recommended for Forge)
6. **Releases** on GitHub → **Draft new release** → tag `v1.0.0`
7. Attach zip from: `..\GamePanelHUDHitFikaFix-release\GamePanelHUDHitFikaFix_v1.0.0.zip`
8. Publish on [The Forge](https://forge.sp-tarkov.com/) — see [docs/FORGE_PUBLISH.md](FORGE_PUBLISH.md)

## Folder layout

| Path | Purpose |
|------|---------|
| `GamePanelHUDHitFikaFix/` | **This repo** — source, docs, push to GitHub |
| `GamePanelHUDHitFikaFix-release/` | **Release only** — zip for players (do not need separate git repo) |

# Русский

1. **GitHub Desktop** → **Файл** → **Добавить локальный репозиторий**
2. Папка: `...\github-repos\GamePanelHUDHitFikaFix`
3. Если «не git-репозиторий» → **создать репозиторий** в этой папке
4. Первый коммит: `Initial release v1.0.0 — Fika hit marker fix for GamePanelHUD`
5. **Опубликовать репозиторий** (public — удобно для Forge)
6. На GitHub → **Releases** → тег `v1.0.0` → прикрепить zip из `..\GamePanelHUDHitFikaFix-release\`
7. Публикация на [The Forge](https://forge.sp-tarkov.com/) — см. [docs/FORGE_PUBLISH.md](FORGE_PUBLISH.md)

# GamePanelHUD Hit Fika Fix



**GitHub:** [kabzon93region](https://github.com/kabzon93region)

**Клиентский мод для SPT 4 + Fika.** Фикс GamePanelHUD Hit для корректной работы в Fika headless на клиенте.



## Требования



- **GamePanelHUD** — оригинальный мод

- **Fika** headless-coop

- **SPT**: 4.0.x

- **BepInEx**: 5.4.x



## Установка



1. Скопировать GamePanelHUDHitFikaFix.dll в BepInEx/plugins/



## Совместимость



- headless_client — ставится на клиент
